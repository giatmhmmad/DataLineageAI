Define data lineage relationships between tables

CSV Format Requirements

Your CSV file must contain the following columns:

  • Job Name: The name of the data processing job/pipeline
  • Target Table Name: The destination table that receives data
  • Source Table Names: The source table(s) that provide data (comma-separated for multiple sources)

Important Notes:

  • All table names must already exist in the system
  • Multiple source tables can be listed separated by commas
  • Spaces around commas will be automatically trimmed

Example CSV content:

Job Name,Target Table Name,Source Table Names
ETL_Process_1,sales_summary,"orders, customers"
Data_Pipeline_2,user_analytics,users
Complex_ETL,final_report,"sales_summary, user_analytics, inventory"
